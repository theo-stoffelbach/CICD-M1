# Wordle Frontend React

Frontend React + Vite pour l'application Wordle.

## Commandes

```bash
npm install
npm run dev
npm run lint
npm run build
```

Par défaut, le frontend appelle l'API sur le même hostname que la page, port `8000`.
Pour changer l'URL :

```bash
VITE_API_URL=http://localhost:8000 npm run dev
```
